# PikPak Assistant
修改自-- [PikPak 批量番号重命名助手](https://greasyfork.org/zh-CN/scripts/549418-pikpak-batch-jav-renamer-assistant))

PikPak 网页端批量处理助手，专注三件事：

- 扫描番号并重命名
- 独立清理小于 `100MB` 的普通文件
- 递归拆分子文件夹，把所有层级的文件搬到最外层

- 重命名前会先预览，再确认执行
- 小文件清理和番号扫描是独立功能
- 拆分子文件夹会递归处理所有层级
- 遇到同名冲突会跳过，不强行覆盖

## 安装

1. 打开 `PikPak Assistant.user.js`
2. 用油猴 / Tampermonkey 安装
3. 进入 `mypikpak.com`、`mypikpak.net` 或 `pikpak.me`

也可以直接在仓库里查看脚本文件：

- [PikPak Assistant.user.js](./PikPak%20Assistant.user.js)

## 使用

### 1. 扫描番号并重命名

1. 选中文件或文件夹
2. 点击 `扫描番号`
3. 扫描完成后进入预览
4. 确认后点 `确认重命名`

### 2. 清理小文件

1. 选中文件夹
2. 点击 `清理小文件`
3. 脚本会递归扫描
4. 找出并删除所有小于 `100MB` 的普通文件

### 3. 拆分子文件夹

1. 选中文件夹
2. 点击 `拆分子文件夹`
3. 脚本会扫描所有层级子文件夹
4. 把文件搬到最外层母文件夹
5. 空子文件夹自动删除

## 配置

- `在文件名开头增加发行日期`
- `修复文件扩展名`

## 注意

- 所有删除和移动操作都会真实执行
- 小文件清理会进回收站
- 拆分子文件夹遇到冲突文件会直接跳过
- 如果没识别到番号，通常是文件名规则不匹配
