# PikPak Assistant 使用说明

## 简介

这是一个运行在 PikPak 网页里的油猴脚本，用来批量处理文件：

- 扫描番号并重命名
- 单独清理小于 100MB 的普通文件
- 单独拆分子文件夹，把所有层级子文件夹里的文件搬到最外层母文件夹

## 使用前

- 先在 `mypikpak.com` / `mypikpak.net` / `pikpak.me` 页面打开网盘
- 用油猴或类似扩展安装这个脚本
- 选中文件或文件夹后，再点对应功能按钮

## 功能说明

### 1. 扫描番号并重命名

1. 选中文件或文件夹
2. 点 `扫描番号`
3. 等脚本扫完后，会进入重命名确认预览
4. 确认无误后点 `确认重命名`

说明：

- 只会处理识别出番号的文件
- 可在配置里选择是否自动补扩展名、是否加发行日期前缀

### 2. 清理小文件

1. 选中文件夹
2. 点 `清理小文件`
3. 脚本会递归扫描当前选中内容
4. 找出所有小于 `100MB` 的普通文件
5. 确认后删除到回收站

说明：

- 这是独立功能，不依赖番号扫描
- 会跳过文件夹，只处理普通文件

### 3. 拆分子文件夹

1. 选中文件夹
2. 点 `拆分子文件夹`
3. 脚本会递归扫描所有层级子文件夹
4. 把子文件夹里的文件全部搬到最外层母文件夹
5. 空的子文件夹会删除

说明：

- 遇到同名冲突文件会直接跳过
- 不是只扫第一层，而是会把所有层级都展开

## 配置选项

- `在文件名开头增加发行日期`
- `修复文件扩展名`
- 排序方式和排序方向

## 注意事项

- 批量操作前建议先看预览
- 删除和移动都是实际操作，执行后不可逆风险较高
- 如果某些文件没被识别，通常是番号解析不到
