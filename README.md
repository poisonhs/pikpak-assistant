# PikPak Assistant

PikPak 网页端批量处理助手。

## 亮点

- 扫描番号并批量重命名
- 独立清理小于 `100MB` 的普通文件
- 递归拆分子文件夹，把文件全部搬到最外层
- 遇到同名冲突会自动跳过
- 重命名前先预览，再确认执行

## 安装

1. 打开 [PikPak Assistant.user.js](./PikPak%20Assistant.user.js)
2. 用油猴 / Tampermonkey 安装
3. 打开 `mypikpak.com`、`mypikpak.net` 或 `pikpak.me`

## 使用

### 扫描番号

1. 选中文件或文件夹
2. 点击 `扫描番号`
3. 预览结果后确认重命名

### 清理小文件

1. 选中文件夹
2. 点击 `清理小文件`
3. 脚本会递归扫描并删除小于 `100MB` 的普通文件

### 拆分子文件夹

1. 选中文件夹
2. 点击 `拆分子文件夹`
3. 所有层级的文件会搬到最外层母文件夹
4. 空子文件夹会自动删除

## 配置

- `在文件名开头增加发行日期`
- `修复文件扩展名`
- 排序方式 / 排序方向

## 注意

- 删除和移动都是实际执行
- 小文件会进回收站
- 冲突文件会跳过
- 没识别到番号时，通常是文件名规则不匹配
