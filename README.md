# PikPak Assistant

PikPak 网页端批量处理助手，专门用来处理番号重命名、小文件清理和子文件夹拆分。

修改自- [PikPak 批量番号重命名助手](https://greasyfork.org/zh-CN/scripts/549418-pikpak-batch-jav-renamer-assistant)

## 重命名亮点

- 增加关键词

```text
番号 [关键词][关键词][关键词] 标题.ext
```

示例：

```text
ABP-123 [中出][NTR][OL] 标题.mp4
```

## 安装

1. 打开 [PikPak Assistant.user.js](./PikPak%20Assistant.user.js)
2. 用油猴 / Tampermonkey 安装
3. 打开 `mypikpak.com`、`mypikpak.net` 或 `pikpak.me`

## 使用

### 1. 扫描番号并重命名

1. 选中文件或文件夹
2. 点击 `扫描番号`
3. 等待扫描完成
4. 在预览里确认新文件名
5. 点击 `确认重命名`

### 2. 清理小文件

1. 选中文件夹
2. 点击 `清理小文件`
3. 脚本会递归扫描所有层级
4. 找出所有小于 `100MB` 的普通文件
5. 确认后移入回收站

### 3. 拆分子文件夹

1. 选中文件夹
2. 点击 `拆分子文件夹`
3. 脚本会递归扫描所有层级子文件夹
4. 把子文件夹里的文件全部搬到最外层母文件夹
5. 空子文件夹自动删除
6. 同名冲突文件自动跳过


## 注意

- 删除和移动都会真实执行
- 小文件清理是独立功能，不依赖扫描番号
- 拆分子文件夹是独立功能，不依赖扫描番号
- 遇到冲突文件会跳过，不会覆盖原文件
- 如果没有识别到番号，通常是文件名规则不匹配
- 如果标题里没有直接标签，脚本会继续尝试从标题正文补关键词
